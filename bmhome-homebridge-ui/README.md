# BMHome Homebridge UI assets

Drop the `homebridge-ui/public` folder into the BMHome plugin repository.

Expected structure:

```text
homebridge-ui/
  public/
    index.html
    apple-touch-icon.png
    favicon.ico
    logo-512.png
    logo.jpeg
    logo.svg
```

The page is designed to mirror the GivHome setup flow while using BMHome wording, branding and HomeKit-first feature toggles.
